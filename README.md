# Hangman Game (Python GUI)
A GUI-based Hangman game developed using **Python** and **Tkinter**.  
The game allows users to guess letters to uncover a hidden word while receiving hints and visual feedback through a graphical interface.

## Features
- Interactive **GUI interface**
- **Random word generation** with hints
- **Input validation** for user guesses
- **Dynamic hangman drawing** showing remaining tries
- **Win/Loss detection system**
- Game automatically **resets after completion**

## Tech Stack
- Python
- Tkinter (GUI Library)
- Random module

## How the Game Works
1. A random word and its hint are selected from the dictionary.
2. The player guesses one letter at a time.
3. If the letter exists in the word, it is revealed in the correct position.
4. If the guess is incorrect, the number of remaining tries decreases and the hangman drawing updates.
5. The player wins when the entire word is guessed correctly before running out of tries.
